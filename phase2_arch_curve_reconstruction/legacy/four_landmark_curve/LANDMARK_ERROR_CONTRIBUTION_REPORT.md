# 四类牙齿 Landmark 对预测误差贡献的量化与结果分析

## 1. 分析目的

本项目为每颗缺失牙预测四个三维 landmark：

- Mesial（近中点）
- Distal（远中点）
- OuterPoint（按当前数据命名，通常对应颊侧/唇侧点）
- InnerPoint（按当前数据命名，通常对应舌侧/腭侧点）

本文希望回答两个不同的问题：

1. 对某一颗预测牙，四个 landmark 分别对牙齿中心误差贡献了多少？
2. 在整个数据集上，是否存在某一种 landmark 持续比其他类型更容易产生误差或主导中心误差？

这两个问题不能仅靠一个指标回答。单颗牙的中心误差分解是确定性的，但要判断某类 landmark 是否具有系统性问题，必须进行跨患者、跨牙位的总体统计。

> 注意：`OuterPoint` 和 `InnerPoint` 是否严格等价于 buccal 和 lingual，应以原始标注规范为准。对前牙而言，outer 更常对应 labial；对上颌而言，inner 更常称 palatal。下文为便于讨论，将它们统称为颊/唇侧点和舌/腭侧点。

## 2. 当前模型与误差定义

设第 \(i\) 个 landmark 的真实坐标和预测坐标分别为：

\[
\mathbf{g}_i \in \mathbb{R}^3,\qquad
\mathbf{p}_i \in \mathbb{R}^3,\qquad i=1,\ldots,4
\]

单个 landmark 的三维误差向量为：

\[
\mathbf{e}_i=\mathbf{p}_i-\mathbf{g}_i
\]

其误差幅值为：

\[
d_i=\|\mathbf{e}_i\|_2
\]

预测中心和真实中心都是四个 landmark 的算术平均：

\[
\mathbf{c}_{pred}=\frac{1}{4}\sum_{i=1}^{4}\mathbf{p}_i,\qquad
\mathbf{c}_{gt}=\frac{1}{4}\sum_{i=1}^{4}\mathbf{g}_i
\]

因此中心偏移向量为：

\[
\Delta\mathbf{c}
=\mathbf{c}_{pred}-\mathbf{c}_{gt}
=\frac{1}{4}\sum_{i=1}^{4}\mathbf{e}_i
\]

中心误差为：

\[
E_c=\|\Delta\mathbf{c}\|_2
\]

四个 landmark 在中心计算中具有完全相同的固定权重 \(1/4\)。模型和中心定义本身没有预设 Mesial、Distal、OuterPoint 或 InnerPoint 中的任何一个更重要。

## 3. 中心误差的精确贡献分解

### 3.1 构建方法

当 \(E_c>0\) 时，先定义最终中心偏移的单位方向：

\[
\mathbf{u}=\frac{\Delta\mathbf{c}}{\|\Delta\mathbf{c}\|_2}
\]

第 \(i\) 个 landmark 对最终中心误差的带符号贡献定义为：

\[
q_i=\frac{1}{4}\mathbf{e}_i\cdot\mathbf{u}
\]

其中：

- \(q_i>0\)：该 landmark 将中心推向最终误差方向；
- \(q_i<0\)：该 landmark 的误差抵消了其他 landmark 造成的部分中心偏移；
- \(q_i\approx0\)：其误差在最终中心偏移方向上的分量较小。

该方法满足：

\[
\sum_{i=1}^{4}q_i
=\left(\frac{1}{4}\sum_{i=1}^{4}\mathbf{e}_i\right)\cdot\mathbf{u}
=\Delta\mathbf{c}\cdot\mathbf{u}
=E_c
\]

因此，Streamlit 中四根柱子的带符号数值之和严格等于中心误差。这是一个精确的几何分解，不是近似解释。

贡献百分比可写为：

\[
s_i=100\%\times\frac{q_i}{E_c}
\]

四项带符号百分比之和为 100%。由于可能存在相互抵消，单项可以为负数，也可能超过 100%，所以它不应被当作普通的非负“责任比例”。

### 3.2 为什么还要同时查看 landmark 误差幅值

\(d_i\) 和 \(q_i\) 回答的是不同问题：

- \(d_i\)：这个 landmark 本身离真实位置有多远；
- \(q_i\)：这个偏差有多少沿着最终中心误差方向传递到了中心。

一个 landmark 可以有很大的 \(d_i\)，但如果误差方向与中心偏移方向近似垂直，它的 \(q_i\) 仍然很小。反过来，一个误差幅值不是最大的 landmark，也可能由于方向高度一致而成为最大的中心误差贡献者。

因此，不能只依据贡献柱状图判断哪个 landmark 的预测质量最差。至少应同时报告：

| 指标 | 含义 | 是否带方向 |
|---|---|---|
| \(d_i=\|\mathbf e_i\|\) | 单个 landmark 定位误差 | 否 |
| \(q_i=(\mathbf e_i\cdot\mathbf u)/4\) | 对中心误差的净贡献 | 是 |
| \(|q_i|\) | 沿中心误差轴的作用强度 | 否，但丢失抵消信息 |

## 4. 对“逐样本结果毫无规律”的原因分析

### 4.1 这种现象在数学上是合理的

牙齿中心误差是四个三维误差向量的向量平均，而不是四个误差幅值的简单平均。最终结果同时取决于每个 landmark 的：

1. 误差大小；
2. 三维误差方向；
3. 与其他三个误差向量的相互增强或抵消。

即使相邻牙齿的四个 \(d_i\) 很接近，只要误差方向不同，最大的 \(q_i\) 就可能完全不同。因此，“相邻牙的最大贡献 landmark 不一致”本身并不能证明计算无规律或模型异常。

### 4.2 当前训练目标没有制造固定的主导 landmark

训练中的 point loss 对四种 landmark 一视同仁；center loss 约束四点平均值；geometry loss 约束牙内六组点间距离。模型一次性联合输出四个点，而不是分别训练四个互不相关的预测器。

这意味着：

- 四种 landmark 没有不同的显式损失权重；
- 一个点的偏差可以通过其他点的反向偏差部分补偿 center loss；
- geometry loss 会耦合四个点，使某个点的调整同时受整体牙冠几何形状约束；
- 模型优化的是总体平均表现，不保证每颗牙都由同一种 landmark 主导误差。

因此，若数据本身不存在稳定的单类标注偏差，模型也没有理由在所有样本中产生同一个主导 landmark。

### 4.3 牙位和形态差异会改变预测难度

不同牙位的牙冠形态、尺寸和四个 landmark 的空间关系不同。前牙、前磨牙和磨牙不能被视为完全同分布的目标。即便是相邻牙，也可能因为以下因素表现不同：

- 牙冠形态复杂度不同；
- 左右侧镜像关系；
- 上颌和下颌差异；
- 邻牙缺失组合不同；
- 可观察牙齿数量和分布不同；
- 个体牙弓形态和局部拥挤程度不同。

Transformer 使用整条牙弓中仍可观察的牙齿作为上下文。改变遮挡组合后，同一目标牙的输入条件也改变，因此主导误差项不是目标牙位的固定属性。

### 4.4 标注与数据提取噪声可能因 landmark 类型和样本而异

原始 landmark 会先匹配到最近的网格顶点，再根据该顶点的牙齿标签归属到具体 FDI。重复标注会取平均。以下因素都可能引入不稳定性：

- 人工标注点在曲面上的位置差异；
- 网格分辨率和扫描质量差异；
- landmark 到最近顶点的吸附误差；
- 牙齿分割边界或顶点标签错误；
- 同一类别存在多个标注时，平均值掩盖了标注分散程度；
- 某些 landmark 的解剖定义比其他点更模糊。

如果噪声是病例相关而不是固定类别相关，就会呈现“有时一个点大，有时多个点大”的结果。

### 4.5 小中心误差时，贡献百分比容易显得不稳定

当四个误差向量大幅互相抵消时，\(E_c\) 会很小。此时：

- 最终方向 \(\mathbf u\) 对微小坐标变化敏感；
- \(s_i=q_i/E_c\) 因分母很小而出现很大的正负百分比；
- landmark 本身可能都存在明显误差，但中心恰好较准确。

因此，对 \(E_c\) 很小的样本不宜重点比较“贡献百分比”。应优先看毫米值 \(q_i\)、landmark 幅值 \(d_i\) 和抵消程度。

可以定义抵消指数：

\[
CI=1-\frac{E_c}{\frac{1}{4}\sum_{i=1}^{4}d_i}
\]

理论上 \(0\le CI\le1\)：

- \(CI\approx0\)：四个误差方向较一致，landmark 偏差直接传递到中心；
- \(CI\approx1\)：误差之间存在强烈抵消，中心误差不能代表单点预测质量。

### 4.6 当前梯度敏感度不是输出误差责任

Streamlit 的高级分析计算目标中心误差相对于“已观察输入 landmark 坐标”的局部梯度。它回答的是：

> 在当前输入附近，轻微改变哪个已观察 landmark，目标牙中心误差变化最快？

它不回答：

> 目标牙的四个输出 landmark 中，哪个实际造成了当前中心误差？

梯度敏感度是局部模型敏感性，不是因果归因，也不是误差的精确分解。不能把高级热力图和四个输出 landmark 的 \(q_i\) 混为同一种“贡献度”。

## 5. 对当前观察结果的解释

目前观察到的现象更支持以下结论：

1. **不存在肉眼可见的、跨所有样本都固定不变的单一主导 landmark。**
2. **中心误差通常是四个三维误差向量共同作用的结果，并包含增强和抵消。**
3. **主导贡献会随患者、牙位、颌位和遮挡上下文变化。**
4. **单颗牙的最大贡献项不稳定，不等于总体上四类 landmark 完全没有统计差异。**

最后一点尤其重要。查看几个 Streamlit 样本只能否定“极强且一致的单点主导模式”，尚不能证明四类 landmark 在总体误差分布上完全相同。是否存在较弱但稳定的类别差异，需要完整测试集统计和置信区间。

## 6. 建议的总体量化方案

### 6.1 分析单位与数据划分

应只在未参与训练的患者级测试集上分析，避免同一患者上下颌跨越训练集和测试集。对每个测试牙弓使用固定、可复现的遮挡方案，并记录：

- patient ID；
- upper/lower；
- FDI 牙位；
- 牙齿分组（前牙、前磨牙、磨牙）；
- 遮挡数量和遮挡组合；
- 四个 \(d_i\)、\(q_i\)、\(|q_i|\)；
- \(E_c\)、平均 landmark error 和 \(CI\)。

患者应作为统计聚类单位，不能把同一患者产生的大量遮挡结果全部当作相互独立样本。

### 6.2 每类 landmark 的核心统计量

对每一种 landmark 建议报告：

1. \(d_i\) 的均值、中位数、标准差、四分位距和 95% bootstrap 置信区间；
2. \(q_i\) 的均值和正负分布，用于识别系统性方向贡献；
3. \(|q_i|\) 的均值和中位数，用于衡量沿中心偏移轴的作用强度；
4. 成为“最大 \(d_i\)”的样本比例；
5. 成为“最大正 \(q_i\)”的样本比例；
6. 在高误差样本（例如 \(E_c\) 前 10%）中的上述统计；
7. 按 jaw、牙齿类型、左右侧和缺失数量分层后的统计。

若四类 landmark 成为最大贡献者的比例都接近 25%，且置信区间高度重叠，则支持“没有固定主导类型”。若某一类在不同分层中仍持续显著高于其他类，才支持系统性主导结论。

### 6.3 推荐的统计检验

由于同一颗牙同时产生四类配对观测，不能使用四组独立样本检验。推荐：

- 总体比较：Friedman 配对非参数检验；
- 事后两两比较：Wilcoxon signed-rank，并进行 Holm 多重比较校正；
- 效应量：同时报告配对差值及 bootstrap 95% CI，避免只看 p 值；
- 更完整的建模：以 landmark 类型、jaw、牙齿分组和遮挡数量为固定效应，以 patient 为随机效应建立混合效应模型。

应分别对 \(d_i\) 和 \(|q_i|\) 检验。二者含义不同，不能只选其中一个。

### 6.4 稳定性分析

要判断“同一牙位的主导 landmark 是否会随上下文变化”，可对同一目标牙重复生成多种合法遮挡组合，计算：

- 最大贡献 landmark 的一致率；
- 四类贡献排序的 Kendall 一致性；
- 每类 \(q_i\) 和 \(d_i\) 的组内变异；
- 同一目标牙在不同遮挡下的中心误差方差。

如果同一目标牙在不同遮挡组合下主导项频繁变化，说明归因强烈依赖上下文，而不是该牙位固有地由某一 landmark 决定。

## 7. 图表建议

总体报告不应只展示若干单样本柱状图。建议至少加入：

1. 四类 \(d_i\) 的配对箱线图或 violin plot；
2. 四类 \(|q_i|\) 的配对分布图；
3. 各 landmark 成为最大误差/最大贡献者的比例和 95% CI；
4. jaw × 牙齿类型的分层热力图；
5. \(E_c\) 与平均 landmark error 的散点图，以颜色表示 \(CI\)；
6. 高中心误差病例的四个误差向量方向图；
7. 同一目标牙在不同遮挡组合下贡献排序的稳定性图。

## 8. 结论

当前 Streamlit 中使用的带符号投影方法，能够对单颗预测牙的中心偏移进行严格、可加和的四 landmark 分解。它适合回答“当前中心偏移由哪些误差沿最终偏移方向共同形成”，但不能单独代表 landmark 的总体定位质量。

现有样本中最大贡献 landmark 不固定，是可以由向量方向、误差抵消、牙位形态、遮挡上下文、联合训练目标以及标注噪声共同解释的合理现象。它说明误差机制很可能是多因素和病例依赖的，而不是由一个固定 landmark 普遍负责。

因此，现阶段最稳妥的表述是：

> 个例观察未发现统一的单 landmark 主导模式。四类 landmark 对中心误差的贡献具有明显样本依赖性，并受到误差方向和相互抵消的影响。是否存在较弱但稳定的类别差异，仍需在患者级独立测试集上，通过 landmark 误差幅值、带符号中心贡献、绝对贡献、最大贡献频率及分层统计进行验证。
